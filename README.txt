IMAGE REPLACEMENT GUIDE

Replace the files in this folder with your own images.

Required filenames:
- logo.png       = Your real business logo
- camera.jpg     = Professional camera / event production image
- mixer.jpg      = Video mixer / live production image
- drone.jpg      = DJI Air 3S / aerial production image
- project-1.jpg  = Completed project image
- project-2.jpg  = Completed project image
- project-3.jpg  = Completed project image
- project-4.jpg  = Completed project image

IMPORTANT:
Keep the same filenames and the website will automatically use your new images.

You can also change the image filenames inside index.html and js/main.js if required.
Do not replace logo.png with a fake logo. Use your actual business logo.
