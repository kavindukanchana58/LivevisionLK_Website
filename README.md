# LivevisionLK Event Media Website

A professional, colorful, cinematic **static HTML, CSS and JavaScript website** for LivevisionLK Event Live Streaming and Dronography.

## Features

- Premium cinematic design
- Responsive mobile layout
- Event live streaming business website
- Multi-camera video production services
- Full HD / 4K recording services
- Event videography
- Drone photography and videography
- Aerial cinematic production
- Completed projects gallery
- Project detail modal
- Scroll reveal animations
- Mobile navigation
- Call buttons
- WhatsApp buttons
- No PHP
- No MySQL
- No Node.js
- No backend
- GitHub Pages friendly

## Run the Website

This is a completely static website.

### Method 1: Open Directly

Double-click:

```text
index.html
```

### Method 2: GitHub Pages

1. Create a GitHub repository.
2. Upload all project files.
3. Open the repository.
4. Go to **Settings**.
5. Select **Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select the `main` branch.
8. Select `/ (root)`.
9. Click **Save**.

GitHub will provide your live website link.

## Create GitHub Repository

### GitHub Website Method

1. Sign in to GitHub.
2. Click **New repository**.
3. Repository name:

```text
event-media-website
```

4. Choose **Public**.
5. Click **Create repository**.
6. Upload the complete project files.
7. Commit the files.

### Git Command Method

```bash
git init
git add .
git commit -m "Create LivevisionLK website"
git branch -M main
git remote add origin YOUR_GITHUB_REPOSITORY_URL
git push -u origin main
```

## Important

This version is intentionally **static only**.

The Admin Dashboard, Firebase Authentication, Firestore and Firebase Storage have been removed according to the requested architecture.

The Completed Projects section is currently a static gallery. To add a new project, replace one of the project images and update the project title/description in:

```text
index.html
```

and:

```text
js/main.js
```
